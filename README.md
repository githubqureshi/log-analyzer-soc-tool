
# 🔐 Log Analyzer Dashboard (Cybersecurity Project)

This project is a Python-based cybersecurity tool that simulates a SOC (Security Operations Center) system.

## 🚀 Features
- Real-time log monitoring
- Brute force attack detection
- Suspicious IP identification
- Alert system with sound
- GUI-based log analysis
- Attack visualization (graphs)

## 🛠 Tech Used
- Python
- Tkinter
- Matplotlib

## 📌 Use Case
This tool helps simulate how security teams monitor logs and detect attacks in real-time.
